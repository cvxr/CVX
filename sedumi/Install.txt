SeDuMi 1.21 Installation instructions

This version of SeDuMi comes with binaries for recent versions of Matlab in both 32 bit and 64 bit flavour. In case the binaries don't work for your system, try to run install_sedumi. Note that you may have to install a compiler for Matlab first.

If you encounter any problems, you can post your question on the SeDuMi forum at http://sedumi.ie.lehigh.edu
